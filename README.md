# E-Commerce Sales Analysis with Python

## Project Overview
This project analyzes e-commerce sales data to identify revenue trends, top product categories, customer behavior, payment preferences, delivery performance, and review patterns.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Brazilian E-Commerce Public Dataset by Olist from Kaggle.

## Business Questions
1. Which product categories generate the highest revenue?
2. What is the monthly sales trend?
3. Which payment methods are most used?
4. Does delivery performance affect customer review scores?
5. Which regions have the most customers?

## Key Insights
- Revenue is concentrated in specific product categories.
- Credit card is the most popular payment method.
- Delivery delays can negatively affect customer satisfaction.
- Some states and cities have stronger customer demand.

## Recommendations
- Prioritize high-revenue categories.
- Improve delivery performance.
- Target high-demand regions with marketing campaigns.
- Monitor review scores by product category and seller.

## How to Run
1. Clone this repository.
2. Install required libraries.
3. Open the notebook in Jupyter.
4. Run all cells.
